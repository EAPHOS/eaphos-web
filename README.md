## 개발 환경 설정

### 필수 요구사항

- Node.js 22.17.1
- Yarn (권장) 또는 npm

### Yarn 설치 및 설정

1. **Yarn 설치**

   ```bash
   # npm을 통한 설치
   npm install -g yarn

   # 또는 corepack 사용 (Node.js 16.10+)
   corepack enable
   ```

2. **Yarn 버전 확인**

   ```bash
   yarn --version
   ```

3. **Yarn 버전 설정**

   ```bash
   yarn set version 4.1.0
   ```


### 설치 및 실행

1. **의존성 설치**

   ```bash
   yarn install
   # 또는
   npm install
   ```

2. **개발 서버 실행**

   ```bash
   yarn dev
   # 또는
   npm run dev
   ```
